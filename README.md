# crypto-ytt-starter

Klaytn ERC721 BApp

## 사전준비

> 코드내 private key 를 자신의 키로 모두 변경합니다.

## 스마트 컨트랙트 빌드

> truffle migrate --compile-all --reset --network klaytn

## NODE 모듈 설치

> npm install

## frontend 실행

> npm run dev

## 버전정보

> node 버전: 16.13.0

> npm 버전: 8.1.4

> truffle 버전: 5.1.23

> solidity 버전: 0.5.16
